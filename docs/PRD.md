## 一、项目概述

### 1.1 产品定位

国内首个AI赋能的装修新手零门槛一站式服务工具，MVP版本仅聚焦「首次装修刚需新手」，解决装修全流程三大核心痛点：不懂设计、预算超支、没时间盯工地，实现"装修小白不用懂行，也能全程可控不踩坑"的核心价值。

### 1.2 核心验证目标（MVP唯一使命）

- **价值验证**：装修新手愿意为「AI设计+预算实时联动+AI云监工」的全流程工具付费，核心功能可解决用户核心痛点，种子用户满意度≥85%
- **商业验证**：299元/装修全周期的专业版定价，付费转化率≥5%，单用户获客成本≤50元，商业模型可跑通；AI辅助开发实现MVP开发周期≤3.5个月，人力成本降低15%以上

### 1.3 产品核心价值主张

| 价值维度 | 核心承诺 | MVP落地路径 |
|----------|----------|--------------|
| 省心 | 全程AI辅助，零装修门槛 | AI自动翻译行业术语、全流程节点主动提醒、踩坑风险实时预警 |
| 省钱 | 全透明定价，零隐藏增项 | 设计与预算实时联动、预算红线自动预警、材料基础比价 |
| 省时 | 一键生成方案，线上全管控 | 10秒生成全套设计方案与预算、AI实时识别施工进度、无需跑工地 |

### 1.4 目标用户画像（MVP仅聚焦此类用户）

| 用户类型 | 核心特征 | 核心痛点 |
|----------|----------|----------|
| 首次装修新手 | 25-35岁，首次购房，零装修经验，刚需预算，对行业术语、施工流程完全陌生，996工作节奏，可支配时间少 | 1. 怕被坑、怕增项，不懂材料与施工工艺，决策无依据<br>2. 不会做装修设计，看不懂专业图纸<br>3. 预算固定，怕超支，对报价不透明极度敏感<br>4. 没时间跑工地、盯施工，无法实时掌握装修进度 |

## 二、需求范围与边界

### 2.1 功能优先级划分规则

- **P0**：MVP必做核心功能，缺之则核心流程断裂、无法完成核心价值验证，本版本100%完成开发
- **P1**：重要非核心功能，不影响核心流程闭环，MVP版本不开发，V1.1版本迭代
- **P2**：锦上添花功能，完全不影响核心价值验证，MVP版本不开发，后续版本按需迭代

### 2.2 MVP功能范围（P0必做）

本版本仅开发以下6大核心模块，完整覆盖用户从进入产品到完成装修全流程闭环：

1. 用户体系与基础框架模块
2. AI验房模块
3. AI交互式设计与预算实时联动模块（核心壁垒）
4. 施工计划与全流程节点管理模块
5. 云监工与AI进度识别模块
6. 合规与数据安全基础体系

### 2.3 功能边界红线（MVP绝对不做）

- 社区内容、UGC问答、装修干货内容生态
- B端企业版全功能（工长/装修公司入驻）
- 供应链直采、材料商城
- 线上增值服务包交易
- 全家共享视图、多账号管理
- 完整验收指导、整改跟踪闭环
- 第三方登录、游客模式

## 三、核心功能需求详情

### 3.1 用户体系与基础框架模块

#### 功能概述
为用户提供极简的注册登录、个人中心管理、付费解锁全流程服务，是产品的基础框架。

#### 核心功能详情

| 功能点 | 功能描述 | 业务规则 |
|--------|----------|----------|
| 手机号一键注册登录 | 用户仅可通过「手机号+短信验证码」完成注册/登录 | 验证码有效时长5分钟，同一手机号1分钟内仅可发送1次 |
| 极简个人中心 | 个人中心仅保留5个核心入口 | 我的订单、设计方案、装修项目、我的报告、账号设置 |
| 极简付费路径 | 全流程关键节点植入专业版解锁引导 | 仅支持微信支付299元/单装修周期专业版 |

### 3.2 AI验房模块

#### 功能概述
为用户提供AI智能验房服务，用户上传房屋照片/户型图，AI自动识别房屋高频高风险问题，生成专业验房报告。

#### 核心功能详情

| 功能点 | 功能描述 | 业务规则 |
|--------|----------|----------|
| 极简验房信息上传 | 用户仅需2步完成操作：上传户型图+房屋现场照片 | 支持JPG/PNG/PDF格式，单文件≤20M |
| AI房屋问题智能识别 | 基于家装垂直微调的视觉模型 | 识别8类问题：墙面开裂、地面平整度、防水渗漏、门窗密封、水电点位、墙体空鼓、阴阳角垂直、下水通畅 |
| 验房报告生成与下载 | 自动生成标准化验房报告 | 报告生成时间≤10秒 |

### 3.3 AI交互式设计与预算实时联动模块（核心壁垒）

#### 功能概述
本模块是产品核心差异化功能，为用户提供AI一键生成装修设计方案、交互式调整、实时联动预算的全流程服务。

#### 核心功能详情

| 功能点 | 功能描述 | 业务规则 |
|--------|----------|----------|
| 户型图智能识别 | 用户上传户型图，AI自动识别户型尺寸、墙体结构 | 识别准确率≥95%，识别时间≤5秒 |
| AI平面布局方案生成 | 基于识别的户型，AI自动生成3套差异化平面布局方案 | 极致空间利用率、亲子友好型、极简通透型 |
| 装修风格选择与效果图生成 | 用户选择装修风格，AI10秒内生成高清全屋效果图 | MVP仅开放3种风格：现代简约、北欧、新中式 |
| 交互式设计调整 | 用户可调整空间布局、主材材质、家具款式 | 调整后实时同步更新效果图与预算，秒级响应 |
| 核心施工图纸生成 | 效果图定稿后，AI自动生成施工必备核心图纸 | 平面布置图、拆改图、水电点位图、开关插座定位图、地面铺装图、材料清单表 |
| 设计-预算实时联动引擎 | 建立「设计元素-工序-材料-人工-地区调价」全链路映射模型 | 预算联动响应时间≤300ms |
| 预算红线管控 | 用户设定预算上限，系统自动预警超支风险 | 实时标红超支项，推荐替代方案 |

### 3.4 施工计划与全流程节点管理模块

#### 功能概述
基于用户的设计方案与装修周期，自动生成标准化施工计划，通过节点主动通知、进度可视化解决装修新手不懂施工流程的痛点。

#### 核心功能详情

| 功能点 | 功能描述 | 业务规则 |
|--------|----------|----------|
| 标准化施工计划自动生成 | 根据用户设计方案、装修总周期，自动生成6大核心节点施工排期 | 核心节点：开工准备→水电改造→防水泥瓦→木工→油漆→竣工 |
| 核心节点主动通知 | 6大核心节点开工前、完工后，通过短信+APP内推送双渠道向用户发送提醒 | 通知触达率≥99.9% |
| 极简进度可视化 | 用进度条+节点状态展示施工进度，延期自动预警 | 延期预警准确率≥90% |

### 3.5 云监工与AI进度识别模块

#### 功能概述
用户/工长上传工地现场照片，AI自动识别施工阶段、完成进度，识别施工不规范问题，自动归档工程档案。

#### 核心功能详情

| 功能点 | 功能描述 | 业务规则 |
|--------|----------|----------|
| 工地照片极简上传 | 用户/工长拍照上传工地现场照片 | 支持单张/批量上传，MVP砍掉短视频上传功能 |
| AI施工进度智能识别 | 基于家装垂直微调的视觉模型，自动识别当前施工阶段、完成进度 | 施工进度识别准确率≥95%，识别响应时间≤15秒 |
| 施工不规范异常预警 | AI识别核心施工不规范问题 | 核心异常问题识别准确率≥85% |
| 工程档案自动归档 | 所有施工照片、节点记录、验收报告自动归档 | 档案永久存储 |

### 3.6 合规与数据安全基础体系

#### 功能概述
搭建符合国家法律法规要求的合规与数据安全体系。

#### 核心功能详情

| 功能点 | 功能描述 | 业务规则 |
|--------|----------|----------|
| 合规文档体系 | 生成符合监管要求的用户协议、隐私政策 | 100%符合《个人信息保护法》《网络安全法》 |
| 用户数据全生命周期安全管控 | 建立用户数据收集、传输、存储、使用、销毁全流程安全管控体系 | 核心敏感数据采用AES-256加密算法加密存储 |
| 基础安全防护 | 服务器安全加固、漏洞扫描、访问控制 | 每周开展一次全量漏洞扫描 |

## 五、核心业务流程

### 5.1 核心用户主流程（MVP全闭环）

```
用户进入APP → 微信/手机号一键登录 → 3步新手引导 → 5题极简装修需求问卷
    ↓
首页核心功能入口 → 分岔路径匹配用户装修阶段
    ├→ 毛坯/二手房收房 → AI验房 → 生成验房报告 → 整改建议 → 跳转设计方案
    ├→ 准备装修 → AI设计 → 平面方案 → 效果图 → 施工图 → 实时预算 → 施工计划生成
    ├→ 施工中 → 云监工 → 进度识别 → 节点通知 → 验收指导 → 问题整改跟踪
    └→ 材料采购 → 材料清单 → 全网比价 → 一键下单 → 进场验收
    ↓
订阅付费转化（免费体验1次AI设计/验房 → 解锁全功能专业版）
    ↓
全周期服务跟进 → 竣工交付 → 工程档案归档 → 局部改造/售后增值服务触达
```

### 5.2 付费转化流程

```
免费版用户触发专业版功能 → 弹出付费解锁引导 → 用户点击支付 → 微信支付成功 → 实时解锁专业版权益 → 跳转功能页面
```

## 六、非功能需求

### 6.1 性能需求

| 指标项 | 性能要求 | 备注 |
|--------|----------|------|
| 页面加载时间 | 首屏加载≤2秒，二级页面加载≤1.5秒 | 4G/5G网络环境下 |

| AI功能响应时间 | 验房报告/效果图生成≤10秒，进度识别≤15秒 | 含AI模型处理时间 |
| 系统可用性 | 核心服务可用性≥99.95% | 月度统计 |
| 并发支持 | 支持10万用户并发访问 | 满足上线初期用户规模需求 |

### 6.2 兼容性需求

| 终端类型 | 兼容性要求 |
|----------|------------|
| 微信小程序 | 兼容微信8.0及以上版本，覆盖99%以上活跃微信版本 |
| 移动端APP | 安卓：Android 10.0及以上；iOS：iOS 14.0及以上 |
| H5移动端 | 兼容Chrome、Safari、微信内置浏览器等主流手机浏览器 |

- 用户密码采用bcrypt不可逆加密存储

- 每周开展一次全量漏洞扫描，高危漏洞24小时内完成修复

- 产品全流程符合《网络安全法》《个人信息保护法》《数据安全法》《电子商务法》

- 所有AI生成内容符合国家相关监管要求
